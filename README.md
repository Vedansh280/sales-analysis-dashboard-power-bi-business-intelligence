# sales-analysis-dashboard-power-bi-business-intelligence
📊 Sales Analysis Dashboard | Power BI Business Intelligence Project
📖 Project Description

The Sales Analysis Dashboard is an end-to-end Business Intelligence project developed using Microsoft Power BI to analyze retail sales performance and generate actionable business insights. The project demonstrates the complete data analytics workflow, beginning with raw data preparation and ending with an interactive dashboard designed to support strategic decision-making.

The dataset consists of 9,994 retail transactions spanning January 2016 to December 2019, covering customer information, product categories, sales, profit, discounts, shipping methods, and regional performance. The project focuses on transforming raw transactional data into meaningful insights through data cleaning, modeling, DAX calculations, and visualization.

Data preparation was carried out using Power Query, where data types were corrected, duplicate records removed, customer names split, custom columns created, and filtered datasets prepared for specific analyses. A dedicated Date Table was created to enable robust time intelligence, and relationships were established following best practices for Power BI data modeling.

The analytical layer was built using 12 DAX measures, including Total Sales, Total Profit, Profit Margin, Year-over-Year Growth, Running Total, Average Sales per Customer, and other KPIs that support advanced business analysis. These measures enabled dynamic calculations across different dimensions and time periods.

The final dashboard consists of 20 interactive visualizations distributed across multiple report pages. It provides detailed analysis of customer performance, regional sales, product profitability, category trends, monthly sales, shipping methods, discount impact, cumulative sales growth, and customer segmentation. Interactive slicers allow users to explore the data dynamically and gain deeper business insights.

Key findings from the analysis reveal that the Technology category delivers the highest profitability, while the West region generates the strongest financial performance. The analysis also highlights that excessive discounting significantly reduces profitability, identifies seasonal sales trends, and pinpoints underperforming products and regions that require strategic attention.

This project showcases practical skills in Business Intelligence, Data Cleaning, Data Modeling, DAX, Dashboard Development, KPI Reporting, Data Visualization, and Business Analytics, making it an ideal portfolio project for aspiring Data Analysts and Business Intelligence Analysts.
